TROUBLESHOOTING

"Connexion perdue - page non disponible en cache"
  → La page du formulaire n'est pas cachée
  → Solution: 
    1. Aller online
    2. Visiter /fiches/creer/
    3. Laisser le Service Worker mettre en cache
    4. Puis aller offline

"Fiche créée mais pas visible après"
  → Template pas chargé
  → Solution: 
    1. Vérifier offline-v4.js chargé (DevTools → Network → offline-v4.js)
    2. Vérifier console errors (F12)
    3. Essayer Ctrl+Shift+R pour nettoyer cache browser

"local_id undefined"
  → handleFormSubmit pas appelé
  → Solution:
    1. Vérifier form a un des selectors: [data-fiche-form], form[action*="/fiche/"], form[action*="creer"]
    2. Ou ajouter data-fiche-form au form

"Sync échoue"
  → /api/sync/ endpoint pas trouvé
  → Solution:
    1. Vérifier inspection/urls.py: path('api/sync/', api.sync_endpoint)
    2. Vérifier inspection/api.py: def sync_endpoint(...)
    3. Redémarrer Django
